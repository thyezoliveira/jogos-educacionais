# Plataforma de Jogos Educativos - SMEC Saquarema

Uma plataforma web moderna (SPA) para hospedar e executar jogos educativos desenvolvidos com **Phaser.js**, integrada ao ecossistema Google Workspace da rede municipal de Saquarema.

## 🚀 Tecnologias principais

- **Frontend:** [Next.js](https://nextjs.org/) (React)
- **Engine de Jogos:** [Phaser.js](https://phaser.io/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Tempo Real:** WebSockets (Socket.io)

## 🎯 Objetivo

Desenvolver um ambiente de aprendizagem interativo onde:
- **Professores** podem gerenciar salas, configurar desafios e monitorar o progresso dos alunos em tempo real.
- **Alunos** participam de jogos educativos utilizando seus e-mails institucionais (@smec.saquarema.rj.gov.br) em seus Chromebooks.

## 📁 Estrutura do projeto

- `docs/`: Documentação e planejamento detalhado do projeto.
- `apps/`: (Futuro) Diretório para frontend e backend.
- `games/`: (Futuro) Diretório modular para cada jogo Phaser individual.
